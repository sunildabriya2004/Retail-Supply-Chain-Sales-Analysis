# Retail-Supply-Chain-Sales-Analysis
🔷 Project Title
Retail Supply Chain & Sales Analysis – United States
🔷 📌 Project Overview

This project presents an end-to-end data analysis workflow, starting from raw data cleaning in Python to building an interactive Power BI dashboard.
It focuses on analyzing sales performance, profitability, customer trends, and delivery efficiency to generate actionable business insights.

🔷 🎯 Business Objectives

Analyze overall sales and profit performance

Identify top-performing states and product categories

Evaluate delivery efficiency across regions

Understand customer segment contribution

Track profit margin and operational performance

🔷 🗂️ Dataset Information

Source: Superstore Retail Dataset

Region: United States

Key Columns:

Order Date

Ship Date

Sales

Profit

Customer Segment

State

Category

🔷 🛠️ Tools & Technologies Used

Python (Pandas, NumPy)

Data Cleaning & Transformation

Exploratory Data Analysis (EDA)

Power BI

DAX

Data Visualization

🔷 ⚙️ Data Analysis Workflow
🔹 1. Data Collection

The raw dataset was gathered and imported into Python for preprocessing.

🔹 2. Data Cleaning

Handled missing values

Removed duplicates

Corrected data types

Created delivery days column

🔹 3. Feature Engineering

Created new business metrics:

Profit Margin

Total Customers

Delivery Performance Metrics

🔹 4. Exploratory Data Analysis (EDA)

Performed EDA to identify:

Sales trends

Regional performance

Category-wise profit

Customer segment contribution

🔹 5. Data Visualization

The cleaned data was loaded into Power BI to build an interactive dashboard.

🔷 📊 Dashboard Features

KPI Cards for Sales, Profit, Customers, Orders & Profit Margin

State-wise supply chain performance

Category-wise delivery analysis

Segment-wise delivery distribution

Interactive filters for dynamic analysis

🔷 📈 Key Insights

Identified top-performing states based on delivery efficiency

High sales does not always result in high profit

Certain categories have longer delivery times

Consumer segment contributes the highest sales
